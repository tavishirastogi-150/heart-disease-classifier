# ❤️ Heart Disease Prediction

## 📌 Overview
This project predicts the likelihood of heart disease using machine learning based on patient health data.

## 🚀 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Logistic Regression
- Streamlit

## ⚙️ Workflow
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Scaling
- Model Training
- Evaluation

## 📊 Results
- Accuracy: ~88%
- Evaluated using precision, recall, F1-score

## 📂 Dataset
Heart disease dataset (~300+ records)

## ▶️ Run Project

pip install -r requirements.txt  
python src/train.py  
streamlit run app.py  

## 📁 Structure
- notebook/analysis.ipynb → EDA + visualization  
- src/train.py → training pipeline  
- app.py → UI  

## 📌 Conclusion
The model helps in early prediction of heart disease, assisting healthcare decisions.
